### 设置

如果要在您的系统上启用IUS仓库，请安装**ius-release**包。此包包含IUS的仓库配置和签名公钥。此外，许多IUS中的软件包依赖于[EPEL](http://mirrors.zju.edu.cn/docs/epel/)仓库，因此也要安装**epel-release**包。

#### RHEL/CentOS 7

```bash
yum install \
https://mirrors.zju.edu.cn/ius/ius-release-el7.rpm \
https://mirrors.zju.edu.cn/epel/epel-release-latest-7.noarch.rpm
```

#### 测试

IUS软件包在被发布到主仓库之前，将首先在ius-testing中提供。安装ius-release时，默认情况下ius-testing仓库是禁用的。如果您希望参与测试新的IUS软件包，请启用它。

```bash
yum install yum-utils
yum-config-manager --enable ius-testing
```
