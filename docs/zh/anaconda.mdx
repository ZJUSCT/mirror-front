### 设置软件包镜像

修改用户目录下的 `.condarc` 文件来使用浙大镜像源。

Windows 用户无法直接创建名为 `.condarc` 的文件，可先执行 `conda config --set show_channel_urls yes` 生成该文件之后再修改。

2. 运行 `conda clean -i` 清除索引缓存，保证用的是镜像站提供的索引。

3. 运行 `conda create -n myenv numpy` 测试一下吧。
