> pypi 镜像目前仅为校内用户提供镜像服务。若您的网络环境处于校外， pip 的 HTTP(S) 请求将会被重定向（ 302 ）至清华 TUNA 镜像站。

### 临时使用
```bash
pip install -i https://mirrors.zju.edu.cn/pypi/web/simple some-package
```
注意，`simple` 不能少，若使用 http 镜像需要加上 `--trusted-host mirrors.zju.edu.cn`

### 设为默认
升级 pip 到最新的版本 (>=10.0.0) 后进行配置：

```bash
pip install pip -U
pip config set global.index-url https://mirrors.zju.edu.cn/pypi/web/simple
```
如果您到 pip 默认源的网络连接较差，临时使用本镜像站来升级 pip ：
```bash
pip install -i https://mirrors.zju.edu.cn/pypi/web/simple pip -U
```

### PDM

通过如下命令设置默认镜像：

```bash
pdm config pypi.url https://mirrors.zju.edu.cn/pypi/web/simple
```

### Poetry

通过以下命令为单个项目设置首选镜像：

```bash
poetry source add --priority=primary mirrors https://mirrors.zju.edu.cn/pypi/web/simple/
```

通过以下命令为单个项目设置补充镜像：

```bash
poetry source add --priority=supplemental mirrors https://mirrors.zju.edu.cn/pypi/web/simple/
```

Poetry 尚未支持全局设置镜像。参考 [python-poetry/poetry#1632](https://github.com/python-poetry/poetry/issues/1632)。

使用全局镜像的临时方案是将 Poetry 的安装器切换回 pip，如下所示。但该方式会在将来的版本中停止支持，参考 [python-poetry/poetry#7356](https://github.com/python-poetry/poetry/pull/7356)。

```bash
poetry config experimental.new-installer false
```

### Astral uv

- Linux/macOS/WSL/Unix: 在 `~/.config/uv/uv.toml` 或者 `/etc/uv/uv.toml`
- Windows: 在 `%APPDATA%\uv\uv.toml` 或者 `%PROGRAMDATA%\uv\uv.toml`

填写下面的内容：

```toml
[[index]]
url = "https://mirrors.zju.edu.cn/pypi/web/simple/"
default = true
```

详情参考[官方配置文件文档](https://docs.astral.sh/uv/configuration/files/)。
